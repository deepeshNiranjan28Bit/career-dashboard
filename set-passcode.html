🔒 Set the site-wide passcode

Generate the one-way hash for your dashboard's passcode. Your passcode itself is never saved, sent, or shown anywhere — only the hash is produced, here in your own browser.

The line to paste will appear here.

Pick something strong and non-obvious — the hash is public in your repo, so a weak or common passcode could be guessed offline.

Then:
  1. Copy the whole window.PRESET_LOCK_HASH = "…"; line above.
  2. Open lock-config.js and replace its last line with it.
  3. git add lock-config.js && git commit -m "set passcode" && git push
Now every device is asked for this passcode when it opens the site. To change it later, repeat; to remove it, set the hash back to "".
